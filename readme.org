#+TITLE: CPP Extractor
#+AUTHOR: RJ Sheperd

** Setup

*** Download Repo
#+BEGIN_SRC bash
  git clone https://github.com/rjsheperd/cpp-extractor.git
  git submodule init --update --recursive
#+END_SRC

*** Install Requirements

**** With NIX
https://nixos.org/download.html

**** With [[https://docs.conda.io/en/latest/miniconda.html][Conda]]:
#+BEGIN_SRC bash
  conda env create -f environment.yml
  conda activate
#+END_SRC

**** With pip:
#+BEGIN_SRC bash
  pip install -r requirements.txt
#+END_SRC

** Run

**** With NIX
#+BEGIN_SRC bash
./castxml.sh behave/src/behave/behaveRun.h
#+END_SRC

**** With Conda/pip
#+BEGIN_SRC bash
./castxml.py behave/src/behave/behaveRun.h
#+END_SRC

** XLS Translation

#+BEGIN_SRC bash
  npm i -g xslt3
  xslt3 -xsl:behave-xform.xsl -s:behave.xml -o:fns.xml
#+END_SRC
